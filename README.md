# SDC-KOTLIN - Android Development Projects

A collection of Android (Kotlin) projects organized by module for the Software Development Course.

## Structure

### Module 1: Introduction to Android Development
| Folder | Description |
|--------|-------------|
| HandsOn1_CustomUI | Customizable UI Component Library |
| HandsOn2_MultiLanguage | Multi-Language Support App |
| CaseStudy1_HelloWorld | Hello World Variations (Linear, Relative, Constraint layouts) |
| CaseStudy2_InfoDisplay | Simple Information Display App |

### Module 2: User Interface Design and Layouts
| Folder | Description |
|--------|-------------|
| HandsOn1_DynamicFormBuilder | Dynamic Form Builder (JSON-driven) |
| HandsOn2_DataVisualization | Interactive Data Visualization App (Charts) |
| CaseStudy1_LoginScreen | Login/Registration Screen with Validation |
| CaseStudy2_RecipeApp | Recipe Display App with RecyclerView |

### Module 3: Activity Lifecycle and Navigation
| Folder | Description |
|--------|-------------|
| HandsOn1_FragmentNavigation | Complex Navigation with Fragments |
| HandsOn2_ImplicitIntents | App with Implicit Intents |
| CaseStudy1_QuizApp | Quiz App with Multiple Screens |
| CaseStudy2_MasterDetail | Master-Detail Flow App |

### Module 4: Data Storage and Networking
| Folder | Description |
|--------|-------------|
| HandsOn1_OfflineFirstApp | Offline-First App with Synchronization |
| HandsOn2_RealTimeChat | Real-Time Chat Application (Basic) |
| CaseStudy1_BackgroundTask | Background Task for Data Updates (WorkManager) |
| CaseStudy2_SimpleCameraApp | Simple Camera App (CameraX) |

## Tech Stack
- **Language**: Kotlin
- **Platform**: Android
- **Build**: Gradle (Kotlin DSL)
- **Min SDK**: 24+
